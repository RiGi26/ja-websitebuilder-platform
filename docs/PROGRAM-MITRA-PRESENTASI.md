> Materi presentasi untuk calon mitra. Bahasa sederhana, siap dipakai

---

## Apa itu Program Mitra?

Program Mitra adalah cara Anda mendapatkan penghasilan dengan merekomendasikan
jasa pembuatan website Japan Arena ke teman, kenalan, atau jaringan bisnis Anda.

Setiap kali orang yang Anda ajak jadi memesan website dan membayar,
**Anda mendapat komisi** — dan **teman Anda mendapat harga lebih murah**.
Dua-duanya untung.

---

## Keuntungannya

Besarnya keuntungan mengikuti nilai order teman Anda:

| Nilai Order | Untuk Anda (Mitra) | Untuk Teman Anda (Pembeli) |
|---|---|---|
| **≥ Rp 1.000.000** | Komisi **10%** dari nilai order | Diskon **5%** dari harga website |
| **Di bawah Rp 1.000.000** | Komisi **Rp 50.000** (tetap) | Diskon **Rp 25.000** (tetap) |

Selain itu:

- Komisi dibayar ke rekening pribadi Anda
- Tanpa target, tanpa kewajiban
- Teman Anda tidak perlu kode kupon ribet — diskon otomatis dari link

Contoh nyata:

> **Order besar** — teman Anda memesan website seharga **Rp 3.000.000**:
> - Teman Anda hemat: **Rp 150.000** (diskon 5%) → bayar Rp 2.850.000
> - Anda dapat komisi: **Rp 285.000** (10% dari nilai akhir)
>
> **Order kecil** — teman Anda memesan website seharga **Rp 750.000**:
> - Teman Anda hemat: **Rp 25.000** → bayar Rp 725.000
> - Anda dapat komisi: **Rp 50.000** — dan karena order kecil dibayar lunas
>   di muka, komisi ini **langsung bisa dicairkan**
>
> Satu order saja sudah terasa. Lima order besar = Rp 1.425.000.

## Cara Kerjanya — 4 Langkah

### 1. Daftar (sekali saja)
Hubungi tim Japan Arena. Kami buatkan akun mitra Anda — login, kode referral,
dan link khusus dikirim otomatis ke WhatsApp + email Anda.

### 2. Bagikan link Anda
Anda punya **satu link pendek** dan **satu kode**, contohnya:

```
Link : japanarena-...../r/BUDI4K7
Kode : BUDI4K7
```

Bagikan lewat WA, Instagram, status, atau sebut langsung — di dashboard
ada tombol **"Bagikan via WhatsApp"** yang teksnya sudah jadi, tinggal kirim.

### 3. Teman Anda order seperti biasa
- Klik link Anda → masuk ke website Japan Arena → pilih paket → isi form order.
- Kode Anda **terisi otomatis** di form (atau bisa diketik manual).
- Diskonnya langsung muncul di rincian harga sebelum bayar.

### 4. Komisi masuk ke dashboard Anda
Begitu teman Anda membayar, komisi tercatat otomatis. Anda dapat
notifikasi WhatsApp setiap ada komisi baru — tidak perlu cek-cek manual.

## Status Komisi — Kapan Bisa Dicairkan?

| Status | Artinya |
|---|---|
| 🟡 **Menunggu Pelunasan** | Teman Anda baru bayar DP. Komisi sudah dicatat, menunggu order dilunasi. |
| 🟢 **Bisa Dicairkan** | Order sudah lunas. Komisi masuk saldo Anda, siap ditarik. |
| 🔵 **Dibayar** | Komisi sudah ditransfer ke rekening Anda. |

Catatan: order kecil yang dibayar lunas di muka → komisinya **langsung**
berstatus Bisa Dicairkan, tanpa menunggu.

---

## Cara Mencairkan Komisi

1. Buka **Dashboard Mitra** (link + login dikirim saat pendaftaran).
2. Isi rekening bank Anda di bagian *Rekening Pencairan* (sekali saja).
3. Saat saldo **Bisa Dicairkan ≥ Rp 100.000**, klik **"Ajukan Pencairan"**.
4. Tim Japan Arena transfer manual ke rekening Anda dalam **1–3 hari kerja**.
5. Anda dapat notifikasi WhatsApp begitu dana terkirim.

---

## Yang Bisa Anda Lihat di Dashboard

- **Link & kode referral** + tombol salin dan bagikan WA
- **4 angka ringkas**: jumlah order, komisi pending, saldo bisa dicairkan, total yang sudah dibayar
- **Riwayat komisi** per order (tanggal, nilai order, komisi, status)
- **Riwayat pencairan** ke rekening Anda
- **Pengaturan rekening bank**

Semua transparan — Anda bisa pantau kapan saja dari HP.

## Ketentuan Singkat (Fair Use)

1. Order ≥ Rp 1 juta: komisi dihitung dari **nilai akhir order setelah diskon**.
   Order di bawah Rp 1 juta: berlaku **tarif tetap** (diskon Rp 25.000, komisi Rp 50.000) yang tidak bisa diubah.
2. Kode referral **tidak bisa dipakai untuk order sendiri** (nomor WA/email sama).
3. Jika order **dibatalkan/refund**, komisi order itu hangus.
4. Komisi berlaku untuk **order website baru**. Upgrade dan perpanjangan tahunan belum termasuk.
5. Minimal pencairan **Rp 100.000**, satu pengajuan aktif dalam satu waktu.
6. Japan Arena berhak menangguhkan akun mitra yang terindikasi curang.
